# Nexus Inventory System

A PHP and MySQL based inventory management system with AI-inspired stock forecasting.

## Features

- User authentication
- Product management
- Sales tracking
- Stock monitoring
- Low stock alerts
- AI stock insights and 7-day forecasting
- Dashboard analytics

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Setup

1. Clone the repository
2. Import the database file into MySQL
3. Update config.php database settings
4. Run using XAMPP
